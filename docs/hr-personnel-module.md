# 37. Модуль "HR / Персонал" (обязательный)

## Статус и назначение

Документ фиксирует структуру и поведение раздела **HR / Персонал** в Fibercell Manufacturing MVP.
Используется как источник требований для реализации и приемки.

---

## 37.1 Основные разделы HR

```text
HR / Персонал
├── Сотрудники
├── Карточки сотрудников
├── Документы
├── Отпуска и больничные
├── Обучение и допуски
├── Отчеты
└── Настройки
```

## 37.2 Главная страница HR

Экран "Сотрудники" должен работать как база персонала:

- поиск по ФИО, табельному номеру, должности;
- фильтры по отделу, линии/участку, статусу, должности;
- карточки сотрудников в общем стиле системы.

Поля карточки сотрудника в списке:

- фото;
- ФИО;
- должность;
- табельный номер;
- отдел/линия;
- телефон;
- статус.

Статусы:

- работает;
- отпуск;
- больничный;
- уволен.

## 37.3 Личная карточка сотрудника

По клику на сотрудника открывается личная карточка:

- как отдельная страница или
- как большое модальное окно поверх списка.

### Блоки карточки

1) Фото сотрудника

- отображение фото;
- кнопка "Загрузить фото" (добавить/заменить).

2) Основная информация

- ФИО;
- табельный номер;
- дата рождения;
- телефон;
- адрес;
- должность;
- отдел;
- линия/участок;
- дата приема;
- статус сотрудника.

3) Рабочая информация

- должность;
- разряд/категория;
- руководитель;
- смена;
- график работы;
- испытательный срок;
- дата перевода;
- история изменений.

4) Документы сотрудника

Поддержка загрузки и хранения документов в карточке сотрудника:

- паспорт/удостоверение личности;
- трудовой договор;
- приказ о приеме;
- медосмотр;
- обучение;
- сертификаты;
- допуски;
- заявление;
- прочие файлы.

Поля и действия для документа:

- название;
- тип документа;
- дата загрузки;
- срок действия;
- кто загрузил;
- действия: посмотреть / скачать / заменить / удалить.

5) Отпуска, больничные, отсутствия

- история отпусков;
- история больничных;
- прогулы;
- командировки;
- добавление нового периода отсутствия.

6) Обучение и допуски

- список инструктажей;
- список обучений;
- список сертификатов;
- список допусков;
- предупреждение о скором истечении срока действия.

7) Заметки HR

- внутренние комментарии;
- замечания;
- рекомендации;
- служебная информация.

## 37.4 Отчеты HR

Минимум:

- список сотрудников;
- новые сотрудники;
- уволенные сотрудники;
- документы с истекающим сроком действия.

## 37.5 Настройки HR

Справочники:

- должности;
- отделы;
- линии/участки;
- типы документов;
- статусы сотрудников.
