# Otgruzka Tovara / Fibercell Manufacturing MVP

MVP для управления производственной цепочкой по пропиточной линии:

1. Прием сырья (сетка, стеклоткань, ратл)
2. Пропитка
3. Контроль качества
4. Склад
5. Отгрузка

## Что в MVP сейчас

- Дашборд с ключевыми показателями (рулоны, метры, партии)
- Канбан-цепочка партий по этапам
- Готовая интеграционная заготовка под Firebase Auth + Firestore (`src/firebase.ts`)
- Подготовка к Vercel деплою

## Быстрый старт

```bash
npm install
npm run dev
```

## Firebase переменные

Создай `.env` из `.env.example`:

```bash
cp .env.example .env
```

И заполни `VITE_FIREBASE_API_KEY`.

## Производственная логика (следующий этап)

- Справочник сырья и поставщиков (`SIMO`, др.)
- Карта партии: закупка -> линия -> QC -> склад -> отгрузка
- Нормы расхода пропитки, потери, выход годного
- Акты списания/брака
- Отгрузочные документы и себестоимость
- Ролевой доступ: склад, технолог, ОТК, бухгалтерия, админ
