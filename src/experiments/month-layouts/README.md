# Лаборатория макетов табеля

Прототипы UI для раздела **Табель**. Не влияют на боевой `MonthPage`.

## Открыть локально

```
npm run dev
```

**http://localhost:5173/#/dev/month-layouts**

Сразу на радикальный вариант F:

**http://localhost:5173/#/dev/month-layouts?v=f**

## Две группы

### Компактные (A–E)
Уменьшение текущей панели: toolbar, icon-rail, dock, accordion, FAB.

### Другая архитектура (F–I)
Принципиально другая компоновка:

| ID | Идея |
|----|------|
| **F** | IDE-сайдбар: слева постоянная навигация (бригады/операции), справа таблица |
| **G** | Вкладки бригад: одна бригада = один экран, без фильтров |
| **H** | Полный экран: только таблица + ⌘K палитра команд |
| **I** | Карточки: список сотрудников с timeline дней, не grid |

## URL-параметр

`?v=f` … `?v=i` — запоминает выбранный вариант в hash.
